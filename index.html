What is Css?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describe the HTML elements are to be displayed on scrren,paper, or in other media

  3. Css is not necessary CSS saves a lot of work.It can control the layout of multiply web pages all at once

  4. External stylesheets are stored in a css file.

CSS can be added to HTML documents in 3 way

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files.However,in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements, ID, on the other hand, appiles a style to one unique element. ID is also special in that you can use a special URL to directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be appiled to the element.

There is a wide range of selection available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be appiled.
